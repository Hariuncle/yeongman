- [ ]  **4. 회고를 잘 작성했나요?**

    배운점과 아쉬운점, 느낀점 등이 기록되어 있는지 확인
    - 전체 코드 실행 플로우를 그래프로 그려서 이해를 돕고 있는지 확인
        - 피어 리뷰 때 아쉬운 점, 느낀점, 배운점을 잘 말씀해주셨는데 아쉽게도 회고를 따로 작성하지는 않으셨습니다.
        - 이는 커널 연결이 끊기면서 노트북 작성이 어려웠기 때문이라고 생각합니다.
